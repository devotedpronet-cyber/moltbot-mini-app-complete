# Moltbot Telegram Mini App

This repository hosts the Moltbot Telegram Mini App - a comprehensive dashboard for managing your Moltbot assistant.

## Features

- **Dashboard**: Overview of active services and notifications
- **Services**: Detailed view of all active services
- **Tasks**: Real-time tracking of current tasks
- **Cron Jobs**: Management of scheduled jobs and automation
- **Weather**: Current weather and forecast for Estoril, Portugal
- **Projects**: Tracking of active projects and progress
- **Files**: File explorer functionality

## Architecture

This is a client-side application that connects to your local Moltbot server for real data. The app is designed as a Progressive Web App (PWA) that works seamlessly within Telegram.

## Security

The app connects securely to your local server. All sensitive data remains on your local machine, with the GitHub Pages site serving only the static interface.

## Deployment

This site is automatically deployed to GitHub Pages and connects to your local Moltbot instance.
